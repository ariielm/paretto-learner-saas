# Paretto SaaS

Plataforma multi-tenant de aprendizado progressivo assistido por IA. Versão SaaS do projeto Paretto Learner original.

## Status

Em desenvolvimento inicial (v0.1). Repositório vazio aguardando execução da **spec 000 (Bootstrap)**.

## Documentos de referência

- [`PLANO_ENTREGAS.md`](./PLANO_ENTREGAS.md) — plano de alto nível: escopo v0.1, roadmap pós-v0.1, riscos.
- [`docs/superpowers/specs/2026-05-20-saas-foundation-design.md`](./docs/superpowers/specs/2026-05-20-saas-foundation-design.md) — design fundacional do projeto: estrutura SDD, constitution, standards, backlog, workflow.
- `.specify/constitution.md` — princípios invioláveis (será criado na spec 000).
- `.specify/standards/*.md` — padrões repetíveis (serão criados na spec 000).
- `docs/decisions/` — ADRs (Architecture Decision Records).

## Metodologia

[Spec-Driven Development](https://github.com/github/spec-kit) com spec-kit oficial. Toda feature segue `spec → plan → tasks → implement`. Stack e ferramentas são decididas no `plan.md` da spec 000.

## Idioma

- Conteúdo do app, prompts, mensagens de erro user-facing: **PT-BR**.
- Código (nomes de variáveis, comentários técnicos): **inglês**.
